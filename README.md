# 📊 Customer Base Performance Analysis Dashboard

An interactive Power BI dashboard designed to analyze customer behavior, loan portfolio performance, digital engagement, and business growth opportunities through data-driven insights.

## 🚀 Project Overview

This project focuses on transforming raw customer data into meaningful business intelligence that supports strategic decision-making. Using a dataset containing 500 customer records, the dashboard provides a comprehensive view of customer financial behavior, risk exposure, and cross-selling opportunities.

The goal was to identify key trends, monitor business KPIs, and provide actionable insights through interactive visualizations.

---

## 🎯 Business Objectives

The dashboard was developed to analyze:

* Customer Savings Behaviour
* Loan Portfolio Risk
* Digital Engagement
* Customer Segmentation
* Cross-Sell Opportunities
* KPI-Based Business Performance
